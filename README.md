# LIPAI — Onboarding em Visão Computacional e Deep Learning

Este repositório reúne todas as atividades, estudos dirigidos e projetos desenvolvidos durante o **Onboarding do LIPAI (Laboratório Interdisciplinar de Processamento e Análise de Imagens)**, um treinamento intensivo voltado à formação em **programação em Python, aprendizado de máquina e deep learning aplicado à análise de imagens**, com foco em imagens histológicas.

O onboarding teve duração de **13 semanas**, cobrindo desde os fundamentos de Python até a implementação de **redes neurais convolucionais para classificação e segmentação semântica de imagens**.

---

## Conteúdos Abordados

### Fundamentos

* Python: sintaxe, controle de fluxo, tipos de dados e funções, programação orientada a objetos
* Manipulação de arquivos, pacotes e módulos

### Ciência de Dados e Machine Learning

* NumPy, Pandas e Matplotlib
* Introdução ao aprendizado de máquina
* Preparação e análise de dados
* Algoritmos de classificação supervisionados
* Desafios em Machine Learning

### Deep Learning e Visão Computacional

* Redes neurais artificiais
* PyTorch
* Redes neurais convolucionais (CNNs)
* Data augmentation
* Transfer learning

### Segmentação de Imagens

* Técnicas clássicas de segmentação
* Segmentação semântica com deep learning
* Fully Convolutional Networks (FCN)
* U-Net

---

## Projetos Desenvolvidos

### Projeto 1 — Classificação de Imagens Histológicas

* Implementação de pipelines de classificação com CNNs
* Uso de data augmentation e transfer learning
* Avaliação de desempenho dos modelos

### Projeto 2 — Segmentação Semântica em Imagens Histológicas

* Implementação de arquiteturas **FCN** e **U-Net**
* Treinamento e validação em datasets histológicos
* Avaliação com métricas como **Accuracy, Dice e IoU**
